O que são tags

As tags HTML (HyperText Markup Language) são elementos usados para estruturar e apresentar o conteúdo de uma página da web. Elas são usadas para definir a forma como o navegador da web interpreta e exibe diferentes partes do documento HTML.

As tags HTML são cercadas por colchetes angulares ("<" e ">") e geralmente vêm em pares, com uma tag de abertura e uma tag de fechamento correspondente. A tag de abertura indica o início de um elemento e contém atributos opcionais que fornecem informações adicionais sobre o elemento, como seu estilo ou comportamento. A tag de fechamento indica o fim do elemento.Aqui está uma explicação resumida de cada uma das tags HTML mencionadas:

  • < h1 > até < h6 >: Essas tags são usadas para definir os cabeçalhos de um documento HTML, sendo < h1 > o maior e < h6 > o menor.Elas são usadas para indicar a importância e a hierarquia do conteúdo.

    • < h1 >

    < h2 >

    < h3 >

    < h4 >

    < h5 >
    < h6 >
  • < p >: Essa tag é usada para definir um parágrafo de texto.

    É comumente utilizada para separar blocos de texto em parágrafos distintos.


  • < mark >: Essa tag é usada para destacar parte do texto, geralmente com um fundo amarelo para chamar a atenção do leitor.

  • < small >: Essa tag é usada para indicar que o texto deve ser exibido em um tamanho menor do que o tamanho padrão.

  • < i >: Essa tag é usada para colocar o texto em itálico, indicando ênfase ou estilo alternativo

  • < u >: Essa tag é usada para sublinhar o texto, geralmente indicando que o texto é um link ou um termo importante.

  • < strong >: Essa tag é usada para enfatizar o texto, geralmente exibindo-o em negrito. É usado para destacar palavras-chave ou frases importantes.

  • < ol >: Essa tag é usada para criar uma lista ordenada, onde cada item é numerado.

    1. Item
    2. Item

  • < ul >: Essa tag é usada para criar uma lista não ordenada, onde cada item é marcado com um símbolo ou ponto.

    • Item 1
    • Item 2
    • Item 3

  • < li >: Essa tag é usada para definir um item em uma lista (< ol > ou < ul >).

  • < a >: Essa tag é usada para criar um link (âncora) para outra página ou recurso.

  • < hr >: Essa tag é usada para criar uma linha horizontal, geralmente usada para separar seções de conteúdo.

  • < sub >: Essa tag é usada para exibir o texto como subscrito, posicionando-o um pouco abaixo da linha de base normal.

  • < sup >: Essa tag é usada para exibir o texto como sobrescrito, posicionando-o um pouco acima da linha de base normal.

  • < font >: Essa tag era usada para definir a fonte e as propriedades de estilo do texto, mas seu uso está obsoleto. Recomenda-se utilizar o CSS para estilizar o texto.

  • < del >: Essa tag é usada para indicar texto que foi removido ou deletado.

  • < abbr >: Essa tag é usada para indicar uma abreviação ou uma sigla, geralmente exibindo uma dica de ferramenta com o texto completo quando o usuário passa o cursor sobre ela.

  • < blockquote >:
    É usada para indicar uma seção de texto que é uma citação de outra fonte. Ela é geralmente usada para destacar um trecho longo de texto citado de um autor, livro, artigo, entre outros.
